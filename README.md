Davoid is a **single-binary red team engagement platform** built entirely in Go. 20 offensive modules ‚ÄĒ recon, MITM, C2, post-exploitation, AD, AI ‚ÄĒ all compiled into one executable with a built-in engagement management system that tracks every finding and produces client-ready reports.

No Python. No venv. No dependency hell. One binary.

---

## Why Davoid

Most offensive tools do one thing. You end up juggling a terminal full of separate tools with no shared context ‚ÄĒ Nmap in one window, Responder in another, manual notes in a text file. Davoid fixes that.

- **Single binary.** One `davoid` executable. Cross-compile for any platform. Drop it anywhere, run it.
- **Engagement-first.** Start every op with `davoid new` ‚ÄĒ every module you run logs findings to that engagement automatically.
- **Client-ready reports.** `davoid report` produces structured Markdown (pipe through `pandoc` for PDF).
- **20 native Go modules.** No subprocess spawning for the core logic ‚ÄĒ all compiled in.

## Installation

### Option 1 ‚ÄĒ Docker (recommended)

Inspect the [`Dockerfile`](Dockerfile) before running. No unknown scripts, no root required for OSINT/web/AI modules.

```bash
# Safe mode ‚ÄĒ OSINT, web recon, AI, phishing, auditor (no raw network access)
docker run -it --rm \
  -v davoid-data:/root/.davoid \
  -v $(pwd)/reports:/app/reports \
  ghcr.io/bryanparreira/davoid:latest

# Full mode ‚ÄĒ all modules including sniff & mitm (requires host network)
docker run -it --rm --net=host \
  --cap-add NET_RAW --cap-add NET_ADMIN \
  -v davoid-data:/root/.davoid \
  ghcr.io/bryanparreira/davoid:latest
```

Or with Compose:

```bash
git clone https://github.com/BryanParreira/Davoid.git && cd Davoid

docker compose --profile safe up    # OSINT / web / AI
docker compose --profile full up    # all modules + packet capture
```

## The Engagement System

Every module you run is tied to an **engagement** ‚ÄĒ a named context with a target scope, timeline, and finding log. All findings are stored in `~/.davoid/engagements.db` (SQLite). Reports go to the current directory.

```
davoid new "Acme Corp - External"    ‚Üí  create engagement, set active
davoid list                          ‚Üí  all engagements with finding counts
davoid report                        ‚Üí  Markdown report for active engagement
davoid report <id>                   ‚Üí  report for specific engagement
```

Log a finding from the CLI:

```bash
davoid finding \
  --title    "Kerberoastable SPN found" \
  --severity HIGH \
  --module   ad_ops \
  --target   "svc_backup@corp.local" \
  --desc     "SPN set on service account, ticket offline-crackable" \
  --evidence "GetUserSPNs output..."
```
